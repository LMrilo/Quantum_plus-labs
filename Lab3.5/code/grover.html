Lab 3.5: Grover's Algorithm

Quantum Search and Amplitude Amplification

Time: 60 min Prerequisites: QC14, Lab 3

In Lab 3, we used an Oracle to discover a global property of a function. In this laboratory, we tackle a highly practical problem: Unstructured Search. Imagine trying to find a specific name in a phone book that has been completely scrambled. We will build Grover's Algorithm, which uses quantum interference to find the needle in the haystack quadratically faster than any classical computer.

The Quadratic Speedup

Classically, finding one specific item in an unsorted list of $N$ items requires checking the list $N/2$ times on average, and $N$ times in the worst case ($\mathcal{O}(N)$).

Grover's Algorithm finds the item in roughly $\sqrt{N}$ steps ($\mathcal{O}(\sqrt{N})$).

For a database of 1,000,000 items:
Classical Computer ≈ 1,000,000 checks | Quantum Computer ≈ 1,000 checks

Part A: The Anatomy of a Quantum Search

Grover's Algorithm always follows a specific logical loop:

  1. Initialize: Create an equal superposition of all possible answers.
  2. The Oracle: Flip the phase (make it negative) only for the correct answer.
  3. The Diffuser: Perform an "Inversion about the Mean" to shrink the wrong answers and amplify the correct one.
  4. Repeat: Steps 2 and 3 are repeated $\approx \sqrt{N}$ times.
The 2-Qubit Magic (Step-by-Step)

We will search a database of $N=4$ items (using 2 qubits). The winning item we are looking for is $|11\rangle$. Let's trace the amplitudes (probabilities are the square of the amplitude).

  1. Superposition (H gates):
    $$|\psi_1\rangle = \frac{1}{2}|00\rangle + \frac{1}{2}|01\rangle + \frac{1}{2}|10\rangle + \frac{1}{2}|11\rangle$$
    The average (mean) amplitude is $\frac{1}{2}$. Every state has a 25% chance of being measured.
  2. The Oracle (Marks $|11\rangle$): The Oracle applies a negative phase to the winner.
    $$|\psi_2\rangle = \frac{1}{2}|00\rangle + \frac{1}{2}|01\rangle + \frac{1}{2}|10\rangle \mathbf{-} \frac{1}{2}|11\rangle$$
    The new mean amplitude drops to $\frac{1}{4}$ (because we have three $+0.5$ and one $-0.5$).
  3. The Diffuser (Inversion about the Mean): The Diffuser reflects every amplitude around the new mean ($\frac{1}{4}$). The geometric formula is: $\text{New} = 2 \times \text{Mean} - \text{Old}$.
    • For the wrong answers ($+0.5$): $2(\frac{1}{4}) - \frac{1}{2} = \frac{1}{2} - \frac{1}{2} = \mathbf{0}$.
    • For the winner ($-0.5$): $2(\frac{1}{4}) - (-\frac{1}{2}) = \frac{1}{2} + \frac{1}{2} = \mathbf{1}$.
    $$|\psi_3\rangle = 0|00\rangle + 0|01\rangle + 0|10\rangle + \mathbf{1}|11\rangle$$
Result: In exactly 1 step, the probability of measuring the winner $|11\rangle$ went from 25% to 100%!

Exercise 1: Visualizing the Amplification

We will build the Oracle to mark $|11\rangle$. A gate that applies a negative phase only when both qubits are 1 is the Controlled-Z (CZ) gate.

Instructions:
  1. Put $q_0$ and $q_1$ in an equal superposition using H gates.
  2. The Oracle: Apply a CZ gate connecting $q_0$ and $q_1$. Look at the Q-Sphere: the state $|11\rangle$ is now Red (negative phase).
  3. The Diffuser: This circuit performs the inversion about the mean:
    • Apply H gates to both qubits.
    • Apply X gates to both qubits.
    • Apply a CZ gate between them.
    • Apply X gates to both qubits.
    • Apply H gates to both qubits.
  4. Look at the Probabilities bar chart.
Expected Effect: The bar chart will show exactly 100% probability on the state $|11\rangle$. The algorithm successfully amplified the marked state and suppressed all others to 0.

Part C: Coding Grover's Algorithm (Qiskit)

Exercise 2: Python Implementation Launch Colab Notebook

Now we implement the 2-qubit Grover search in Qiskit to find the state $|11\rangle$.

from qiskit import QuantumCircuit
from qiskit_aer import AerSimulator

# 1. Setup
qc = QuantumCircuit(2, 2)

# 2. Initialization: Equal Superposition
qc.h([0, 1])
qc.barrier()

# 3. THE ORACLE (Marks state |11>)
# A Controlled-Z gate flips the phase only if both qubits are 1
qc.cz(0, 1)
qc.barrier()

# 4. THE DIFFUSER (Amplitude Amplification)
# Step 4a: Apply H gates
qc.h([0, 1])
# Step 4b: Apply X gates (shifts the |00> state to |11>)
qc.x([0, 1])
# Step 4c: Apply CZ (marks the new |11> state)
qc.cz(0, 1)
# Step 4d: Undo X and H
qc.x([0, 1])
qc.h([0, 1])
qc.barrier()

# 5. Measurement
qc.measure([0, 1], [0, 1])

# 6. Execution
simulator = AerSimulator()
job = simulator.run(qc, shots=1000)
print(f"Search Results: {job.result().get_counts()}")
Code Analysis: The output will be 100% {'11': 1000}. You found the hidden data in exactly 1 query! The logic of the Diffuser (Steps 4a-4d) is universally used to amplify probabilities in many other quantum algorithms.
