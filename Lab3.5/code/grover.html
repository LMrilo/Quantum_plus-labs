Lab 3.5: Grover's Algorithm

Quantum Search and Amplitude Amplification

Time: 60 min Prerequisites: QC14, Lab 3

In Lab 3, we used an Oracle to discover a global property of a function. In this lab, we tackle a highly practical problem: Unstructured Search. We will build Grover's Algorithm, which uses quantum interference to find a specific item with a quadratic speedup over classical computing.

The Quadratic Speedup

Classically, finding an item in an unsorted database of $N$ elements requires checking $N/2$ items on average ($\mathcal{O}(N)$). Grover's Algorithm finds it in approximately $\sqrt{N}$ steps ($\mathcal{O}(\sqrt{N})$).

For a database of 1,000,000 items:
Classical Computer ≈ 500,000 checks  |  Quantum Computer ≈ 1,000 iterations

Part A: Mathematical Evolution

Grover's algorithm alternates between two operators: the Oracle $U_\omega$ and the Diffuser $U_s$. Let's formalize the state vector evolution for a general search space of $N$ elements.

1. State Vector Evolution

Step 1 — Initialization. Three Hadamard gates create a uniform superposition of all $N = 2^n$ basis states:

$$ |\psi_0\rangle = |s\rangle = \frac{1}{\sqrt{N}} \sum_{x=0}^{N-1} |x\rangle $$

We decompose $|s\rangle$ in the 2D plane spanned by the target state $|\omega\rangle$ and the uniform superposition of all other states $|s'\rangle$:

$$ |s'\rangle = \frac{1}{\sqrt{N-1}} \sum_{x \neq \omega} |x\rangle $$ $$ |s\rangle = \sin(\theta)\,|\omega\rangle + \cos(\theta)\,|s'\rangle \qquad \theta = \arcsin\!\left(\frac{1}{\sqrt{N}}\right) $$

The individual amplitudes at initialization are therefore:

$$ \alpha_\omega^{(0)} = \sin\theta = \frac{1}{\sqrt{N}}, \qquad \alpha_x^{(0)} = \frac{\cos\theta}{\sqrt{N-1}} = \frac{1}{\sqrt{N}} \quad (x \neq \omega) $$

Step 2 — The Grover Operator $G = U_s\,U_\omega$.

  • The Oracle $U_\omega = I - 2|\omega\rangle\langle\omega|$ reflects the state across the $|s'\rangle$ axis: $U_\omega|\omega\rangle = -|\omega\rangle$ (flip sign of target), $U_\omega|x\rangle = |x\rangle$ for all $x\neq\omega$.
  • The Diffuser $U_s = 2|s\rangle\langle s| - I$ reflects the state across the initial superposition $|s\rangle$.
Why $U_s$ implements "inversion about the mean": If the current amplitudes are $\{\alpha_x\}$ with mean $\mu = \frac{1}{N}\sum_x \alpha_x$, then: $$(U_s|\psi\rangle)_x = 2\langle s|\psi\rangle \cdot \frac{1}{\sqrt{N}} - \alpha_x = 2\mu - \alpha_x$$ Each amplitude is reflected around the mean $\mu$. Large amplitudes get pushed down; below-mean amplitudes get boosted. After the oracle flips the target amplitude negative, $\mu$ drops slightly below the mean of the rest — so inversion about $\mu$ boosts the target dramatically.

Two successive reflections (first across $|s'\rangle$, then across $|s\rangle$) compose into a rotation by $2\theta$ in the 2D plane spanned by $|\omega\rangle$ and $|s'\rangle$. Each application of $G$ rotates the state vector by $2\theta$ towards $|\omega\rangle$. After $k$ iterations:

$$ |\psi_k\rangle = G^k|s\rangle = \sin\!\bigl((2k+1)\theta\bigr)\,|\omega\rangle + \cos\!\bigl((2k+1)\theta\bigr)\,|s'\rangle $$

Expanding $|s'\rangle$, the individual amplitude of each non-target state is:

$$ \alpha_\omega^{(k)} = \sin\!\bigl((2k+1)\theta\bigr), \qquad \alpha_x^{(k)} = \frac{\cos\!\bigl((2k+1)\theta\bigr)}{\sqrt{N-1}} \quad (x \neq \omega) $$
2. Optimal Iteration Count

We maximise $P(\omega) = \sin^2\!\bigl((2k+1)\theta\bigr)$ by setting $(2k+1)\theta = \frac{\pi}{2}$, giving the exact formula:

$$ k^* = \operatorname{round}\!\left(\frac{\pi}{4\theta} - \frac{1}{2}\right) $$

For large $N$, $\theta \approx \sin\theta = 1/\sqrt{N}$, recovering the famous bound $k^* \approx \frac{\pi}{4}\sqrt{N}$.

3-Qubit Application ($N = 8$): $\;\theta = \arcsin\!\bigl(1/\sqrt{8}\bigr) \approx 0.3614\text{ rad}$, $\;k^* = \operatorname{round}\!\bigl(\pi/(4\times 0.3614) - 0.5\bigr) = \operatorname{round}(1.677) = \mathbf{2}$ iterations. Final success probability: $P(\omega) = \sin^2(5\theta) \approx \mathbf{94.5\%}$.

Exercise 1: The Oracle and the Diffuser

We will build the Oracle to mark $|111\rangle$. A gate that applies a negative phase only when all three qubits are $|1\rangle$ is the Multi-Controlled Z (CCZ), implemented as $H$–CCX–$H$ on the target qubit.

Instructions
  1. Apply H to $q_0, q_1, q_2$ to create a uniform superposition.
  2. Oracle block: H on $q_2$ → CCX (ctrl $q_0,q_1$; tgt $q_2$) → H on $q_2$.
  3. Diffuser block:
    • Apply H then X to all qubits.
    • Repeat the CCZ logic (H–CCX–H on $q_2$).
    • Re-apply X then H to all qubits.
  4. Repeat steps 2–3 for the second Grover iteration ($k^* = 2$ for $N = 8$).
Circuit Structure (1 Iteration)

Repeat the Oracle + Diffuser block $k^* = 2$ times.
