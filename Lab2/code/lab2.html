Qiskit uses "Little Endian" ordering. This means the rightmost bit in a string represents qubit 0 ($q_0$).

Exercise 1: The CNOT Gate (Conditional Logic) GATE CNOT
Context [QC05]: The CNOT gate acts on two qubits: a Control and a Target. It flips the Target qubit ONLY IF the Control qubit is in state $|1\rangle$.

CNOT |00⟩ → |00⟩
CNOT |10⟩ → |11⟩
Instructions:
  1. Use an X gate on $q_0$ (Control) to set it to $|1\rangle$. Leave $q_1$ (Target) at $|0\rangle$.
  2. Drag a CNOT gate (symbol ⊕ connected to a dot) into the circuit.
  3. Make sure the dot (control) is on $q_0$ and the plus (target) is on $q_1$.
  4. Observe the Output State in the Probabilities chart.

Expected Effect: The state becomes $|11\rangle$ (Probability 100% on "11"). The target flipped because the control was 1.

Exercise 2: Creating the Bell State (Entanglement)
GATE H + GATE CNOT
Context [QC05]: Entanglement is a state where two particles become inextricably linked. The most famous example is the Bell State $|\Phi^+\rangle$: $$ |\Phi^+\rangle = \frac{|00\rangle + |11\rangle}{\sqrt{2}} $$ This state is created by applying a Hadamard gate followed by a CNOT.
Instructions:
  1. Clear the circuit.
  2. Apply an H gate to $q_0$. (The state is now a superposition of 0 and 1).
  3. Apply a CNOT gate (Control $q_0$, Target $q_1$).
  4. Check the Probabilities.
Q-Sphere Visualization:
Look at the Q-Sphere. You will see two dots connected or symmetric.
Unlike two separate qubits in superposition (which would show 4 dots: 00, 01, 10, 11), an entangled state only shows the correlated outcomes (00 and 11). This visual "sparseness" is a signature of entanglement.

Deep Dive: Why is Entanglement so Important?

2025 Industry Perspective

Entanglement is the key resource for quantum advantage.

  1. Exponential State Space: Scaling the number of qubits increases the computational space as $2^N$.
    • 2 Qubits = 4 states (Simulatable by hand)
    • 53 Qubits ≈ 9 Petabytes (Limit of Classical Supercomputers)
    • 72+ Qubits (e.g., Google Willow) = Beyond classical simulation capabilities.
  2. Connectivity Architectures:
    • Superconducting (e.g., IBM/Google): Typically limited to nearest-neighbor entanglement. Moving information requires SWAP chains.
    • Trapped Ions (e.g., IonQ Tempo): Support "all-to-all" connectivity, allowing direct entanglement between any pair of ions, simplifying algorithm implementation.

Task 4: Scaling Up (GHZ State)

We expand to 3 qubits to create the GHZ State:

$$ |GHZ\rangle = \frac{|000\rangle + |111\rangle}{\sqrt{2}} $$
Concept Preview: Decoherence.
As confirmed by recent benchmarks, maintaining coherence across multiple qubits is exponentially harder due to environmental noise. You might start seeing errors like '001' in real systems!
