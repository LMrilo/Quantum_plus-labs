Exercise 1: The CNOT Gate (Conditional Logic) GATE CNOT
Context [QC05]: The CNOT gate acts on two qubits: a Control ($q_0$) and a Target ($q_1$). Rule: flip the target if and only if the control is $|1\rangle$.

Complete truth table (all 4 basis states):
InputControl $q_0$Target $q_1$OutputAction
$|00\rangle$00$|00\rangle$control=0, no flip
$|01\rangle$01$|01\rangle$control=0, no flip
$|10\rangle$10$|11\rangle$control=1, flip target
$|11\rangle$11$|10\rangle$control=1, flip target
In formula: CNOT$|q_0, q_1\rangle = |q_0,\; q_1 \oplus q_0\rangle$ where $\oplus$ is XOR (addition mod 2). This ensures the gate is reversible.
Instructions:
  1. Use an X gate on $q_0$ (Control) to set it to $|1\rangle$. Leave $q_1$ (Target) at $|0\rangle$.
  2. Drag a CNOT gate (symbol ⊕ connected to a dot) into the circuit.
  3. Make sure the dot (control) is on $q_0$ and the plus (target) is on $q_1$.
  4. Observe the Output State in the Probabilities chart.

Expected Effect: The state becomes $|11\rangle$ (Probability 100% on "11"). The target flipped because the control was 1.

Exercise 2: Creating the Bell State (Entanglement)
GATE H + GATE CNOT
Context [QC05] — Entanglement: A two-qubit state is entangled if it cannot be written as a product $|\psi\rangle \otimes |\phi\rangle$. For example, $\frac{|00\rangle + |11\rangle}{\sqrt{2}}$ cannot be factored — measuring one qubit instantly determines the other.
Step-by-step derivation of $|\Phi^+\rangle$:
  1. Start: $|00\rangle = |0\rangle_{q_0} \otimes |0\rangle_{q_1}$
  2. Apply H to $q_0$: $H|0\rangle_{q_0} \otimes |0\rangle_{q_1} = \frac{|0\rangle+|1\rangle}{\sqrt{2}} \otimes |0\rangle = \frac{|00\rangle + |10\rangle}{\sqrt{2}}$
  3. Apply CNOT ($q_0$ control, $q_1$ target): $$\frac{|00\rangle + |10\rangle}{\sqrt{2}} \xrightarrow{\text{CNOT}} \frac{|00\rangle + |11\rangle}{\sqrt{2}} = |\Phi^+\rangle$$ ($|00\rangle$ stays unchanged; $|10\rangle$ becomes $|11\rangle$ because control=1)
Why is this entangled? If you measure $q_0$ and get 0, the state collapses to $|00\rangle$ — you know $q_1 = 0$ without measuring it. They are correlated.
Instructions:
  1. Clear the circuit.
  2. Apply an H gate to $q_0$. (The state is now a superposition of 0 and 1).
  3. Apply a CNOT gate (Control $q_0$, Target $q_1$).
  4. Check the Probabilities.
The 4 Bell States (complete set):
StateFormulaHow to prepare
$|\Phi^+\rangle$$\frac{|00\rangle+|11\rangle}{\sqrt{2}}$H on $q_0$, then CNOT ← this exercise
$|\Phi^-\rangle$$\frac{|00\rangle-|11\rangle}{\sqrt{2}}$H + CNOT, then Z on $q_0$
$|\Psi^+\rangle$$\frac{|01\rangle+|10\rangle}{\sqrt{2}}$X on $q_0$, then H + CNOT
$|\Psi^-\rangle$$\frac{|01\rangle-|10\rangle}{\sqrt{2}}$X + Z on $q_0$, then H + CNOT
All 4 are maximally entangled and form a complete orthonormal basis for 2-qubit space.
Q-Sphere Visualization:
Look at the Q-Sphere. You will see two dots connected or symmetric.
Unlike two separate qubits in superposition (which would show 4 dots: 00, 01, 10, 11), an entangled state only shows the correlated outcomes (00 and 11). This visual "sparseness" is a signature of entanglement.

Deep Dive: Why is Entanglement so Important?

2025 Industry Perspective

Entanglement is the key resource for quantum advantage.

  1. Exponential State Space: Scaling the number of qubits increases the computational space as $2^N$.
    • 2 Qubits = 4 states (Simulatable by hand)
    • 53 Qubits ≈ 9 Petabytes (Limit of Classical Supercomputers)
    • 72+ Qubits (e.g., Google Willow) = Beyond classical simulation capabilities.
  2. Connectivity Architectures:
    • Superconducting (e.g., IBM/Google): Typically limited to nearest-neighbor entanglement. Moving information requires SWAP chains.
    • Trapped Ions (e.g., IonQ Tempo): Support "all-to-all" connectivity, allowing direct entanglement between any pair of ions, simplifying algorithm implementation.

Task 4: Scaling Up (GHZ State)

We expand to 3 qubits to create the GHZ State:

$$ |GHZ\rangle = \frac{|000\rangle + |111\rangle}{\sqrt{2}} $$

Why does this circuit work?

  1. Start: $|000\rangle$
  2. H on $q_0$: $\frac{|0\rangle+|1\rangle}{\sqrt{2}} \otimes |0\rangle \otimes |0\rangle = \frac{|000\rangle+|100\rangle}{\sqrt{2}}$
  3. CNOT($q_0 \to q_1$): $\frac{|000\rangle+|110\rangle}{\sqrt{2}}$ ($|100\rangle \to |110\rangle$ because $q_0=1$ flips $q_1$)
  4. CNOT($q_1 \to q_2$): $\frac{|000\rangle+|111\rangle}{\sqrt{2}}$ ($|110\rangle \to |111\rangle$ because $q_1=1$ flips $q_2$)
qc = QuantumCircuit(3, 3)
qc.h(0)
qc.cx(0, 1)   # q0 controls q1
qc.cx(1, 2)   # q1 controls q2
qc.measure([0,1,2],[0,1,2])
Concept: Decoherence.
More qubits = more contact with the environment = faster decoherence. On real hardware you may see error states like '001' or '010'.
