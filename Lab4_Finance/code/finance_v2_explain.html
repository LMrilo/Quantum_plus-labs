You built QAOA for MaxCut in Lab 4. This lab applies the same algorithm to a real financial problem: selecting the optimal 2-asset portfolio from {AAPL, MSFT, JPM, XOM}. The quantum circuit changes only in its Hamiltonian — every technique (parameter-shift, COBYLA, QUBO mapping) is a direct carry-over.

 The Scaling Argument

For $n=4$, $K=2$: only 6 portfolios — trivially classical. For $n=50$, $K=10$: $\sim\!10^{10}$ portfolios — classical brute-force infeasible. QAOA scales as $\mathcal{O}(n^2)$ gates per layer; classical optimizers hit an exponential wall. This lab builds the foundations for that future advantage.

import numpy as np, math, itertools
from qiskit.quantum_info import SparsePauliOp
from qiskit_aer.primitives import Estimator, Sampler

# ── Real asset data (AAPL, MSFT, JPM, XOM — annualised 2020-2024) ──────────
MU    = np.array([0.340, 0.270, 0.150, 0.220])
SIGMA = np.array([
    [0.0841, 0.0588, 0.0313, 0.0203],
    [0.0588, 0.0729, 0.0272, 0.0170],
    [0.0313, 0.0272, 0.0576, 0.0294],
    [0.0203, 0.0170, 0.0294, 0.1225]])
TICKERS = ['AAPL', 'MSFT', 'JPM', 'XOM']

def build_qubo(lam=1.0, lam_pen=3.0, K=2):
    n = len(MU)
    Q = lam * SIGMA - np.diag(MU)        # risk-return objective
    P = np.full((n,n), 2*lam_pen)        # penalty off-diagonal
    np.fill_diagonal(P, lam_pen*(1-2*K)) # penalty diagonal
    return Q + P

def portfolio_cost(bits, Q):
    x = np.array(bits)
    return float(x @ Q @ x)

# Enumerate all feasible portfolios (K=2 assets from 4)
Q = build_qubo(lam=1.0)
for combo in itertools.combinations(range(4), 2):
    x = [1 if i in combo else 0 for i in range(4)]
    names = '+'.join(TICKERS[i] for i in combo)
    print(f"{names:15s}  cost={portfolio_cost(x,Q):.4f}")
