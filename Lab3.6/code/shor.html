Lab 3.6: Shor's Algorithm

Quantum Factoring via Period Finding and the QFT

75 min Prerequisites: Lab 3.5, QC14 N = 15, a = 7

Peter Shor's 1994 algorithm can factor an $n$-bit integer in $\mathcal{O}(n^3)$ quantum operations, compared to the classical $\mathcal{O}(e^{1.9n^{1/3}})$. It reduces factoring to period finding, then solves period finding with the Quantum Fourier Transform.

 The Exponential Speedup

RSA-2048 requires factoring a 2048-bit integer. The best classical algorithm (GNFS) would take ~$10^{23}$ operations — longer than the age of the universe on any classical hardware.

Classical (GNFS): $\mathcal{O}\!\left(e^{1.9\,n^{1/3}}\right)$  |  Quantum (Shor): $\mathcal{O}(n^3)$

Part A: Classical Reduction

From Factoring to Order Finding

Step 1. Choose $a$ with $\gcd(a, N) = 1$. If $\gcd > 1$, we found a factor already.

Step 2. Find the multiplicative order $r$: the smallest $r > 0$ with $a^r \equiv 1 \pmod{N}$. This is the period of $f(x) = a^x \bmod N$.

Step 3. If $r$ is even and $a^{r/2}\not\equiv -1 \pmod{N}$, compute:

$p = \gcd(a^{r/2} - 1,\, N), \qquad q = \gcd(a^{r/2} + 1,\, N)$
For $N=15$, $a=7$: $r = 4$  →  $\gcd(7^2-1, 15) = \gcd(48,15) = \mathbf{3}$  and  $\gcd(7^2+1, 15) = \gcd(50,15) = \mathbf{5}$.   $3 \times 5 = 15$ ✓

Part F: Qiskit Python Implementation

Exercise 2: Full Shor Simulation ($N=15$, $a=7$) Open Colab Notebook

The circuit uses 4 counting qubits and a 2-qubit orbit-encoded work register. The modular exponentiation is built from controlled cyclic shift gates.

import math, fractions
from qiskit import QuantumRegister, ClassicalRegister, QuantumCircuit
from qiskit_aer import AerSimulator

def build_qft(n):
    qc = QuantumCircuit(n)
    for i in range(n):
        qc.h(i)
        for j in range(i+1, n):
            qc.cp(math.pi / 2**(j-i), j, i)
    for i in range(n//2):
        qc.swap(i, n-1-i)
    return qc

def build_shor_N15_a7():
    C = QuantumRegister(4, 'count')
    W = QuantumRegister(2, 'work')
    c = ClassicalRegister(4)
    qc = QuantumCircuit(C, W, c)

    qc.h(C)                          # superposition
    qc.barrier()
    qc.ccx(C[0], W[0], W[1])         # c-U^1 (counting[0])
    qc.cx(C[0], W[0])
    qc.cx(C[1], W[1])                # c-U^2 (counting[1])
    qc.barrier()
    qc.compose(build_qft(4).inverse(),
               qubits=C, inplace=True)  # IQFT
    qc.measure(C, c)
    return qc

qc = build_shor_N15_a7()
counts = AerSimulator().run(qc, shots=2000).result().get_counts()

for state, n in sorted(counts.items(), key=lambda x: -x[1]):
    y    = int(state, 2)
    phi  = fractions.Fraction(y, 16).limit_denominator(15)
    r    = phi.denominator
    prob = n/2000*100
    print(f"|{state}> y={y:2d}  phi={phi}  r={r}  ({prob:.1f}%)")

# Factor extraction
r, a, N = 4, 7, 15
print(f"\ngcd(a^(r/2)-1, N) = gcd({a**(r//2)-1}, {N}) = {math.gcd(a**(r//2)-1,N)}")
print(f"gcd(a^(r/2)+1, N) = gcd({a**(r//2)+1}, {N}) = {math.gcd(a**(r//2)+1,N)}")
